# Setup
1. Download as a Zip
2. Unzip and open in VSC
3.  Download Node.js either from
    - https://nodejs.org/en/download/
    - or install manually
      - winget install Schniz.fnm
      - fnm install 22
      - node -v # Should print "v22.15.1".
      - npm -v # Should print "10.9.2".


You need Node.js by running npm init -y
